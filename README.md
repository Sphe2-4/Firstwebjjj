# java_basic_library_system

// how to run locally 

1. you must have a java compiler and java runtime environment installed in your operating system
2. save the code to a single folder all 3 files must be in a single directory 
3. use comand prompt/bash and go to the directory example : cd ~/home/user1/project/java_basic_library_system
4. run the javac command as follows: javac Main.java
5. run the java file as follows: java Main
